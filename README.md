# iOSApp5 - Video & Audio Player Application

## Created By
**Fozia Akhtar**

## Project Description

iOSApp5 is a SwiftUI multimedia application that allows users to browse, play, and share video and audio files stored inside the application bundle.

The app demonstrates important iOS development concepts including media playback, file management, settings management, and dynamic user interface updates.

---

# Features

## 🎬 Video Library

- Automatically loads MP4 video files from the application bundle
- Displays:
  - Video thumbnail
  - Video name
  - Video duration
- Search videos using Searchable
- Navigate to video player screen
- Plays videos using AVPlayer
- Supports video sharing

---

## 🎵 Audio Library

- Automatically loads MP3 audio files from the application bundle
- Displays:
  - Audio name
  - Audio duration
- Search audio files
- Navigate to audio player screen
- Audio controls:
  - Play
  - Pause
  - Stop
  - Volume adjustment
- Supports audio sharing

---

## ⚙️ Settings Management

The application includes a Settings.bundle integration.

Users can configure:

- Theme Mode
  - System
  - Light
  - Dark

- Background Colour
  - White
  - Gray
  - Yellow
  - Blue

- Auto Play option

- Default Volume

Settings are stored using UserDefaults and automatically updated using NotificationCenter.
